Bootstrap Tutorial

Before we can start, we need to include the relevant bootstrap files in our index.html file.

Depending on the version of Bootstrap you use, this will need to be adjusted accordingly.

I have used v4 here.

Getting Bootstrap linked in your project

First, we need to link the CSS file for bootstrap 4. In this case, I will be using a CDN link instead of having bootstrap files downloaded to your local folder.

CSS:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@4.6.2/dist/css/bootstrap.min.css" integrity="sha384-xOolHFLEh07PJGoPkLv1IbcEPTNtaed2xpHsD9ESMhqIYd0nLMwNLD69Npy4HI+N" crossorigin="anonymous">

It is important that you add the bootstrap CSS before any user stylesheet that you may have.

Next, You need to add the following javascript before the closing </body> tag. At the very end, as you can see in this pages index.html file

                <script src="https://cdn.jsdelivr.net/npm/jquery@3.5.1/dist/jquery.slim.min.js" integrity="sha384-DfXdz2htPH0lsSSs5nCTpuj/zy4C+OGpamoFVy38MVBnE+IbbVYUew+OrCXaRkfj" crossorigin="anonymous"></script>
                <script src="https://cdn.jsdelivr.net/npm/bootstrap@4.6.2/dist/js/bootstrap.bundle.min.js" integrity="sha384-Fy6S3B9q64WdZWQUiU+q4/2Lc9npb8tCaSX9FK7E8HnRr0Jz8D6OP9dO5Vg3Q9ct" crossorigin="anonymous"></script>

Now we are ready to use bootstrap. You can add the classes to existing divs right way, but for the purpose of this demo, we need to first understand the grid system.

Bootstraps grid system is built inside a div with the class name "container", and this container class div is necessary to use the grid system in bootstrap

Create a new div with the class container. I'll give all divs a background-color so we can see how it works I placed some dummy p tags in there so you can see it.

We can see that each of the three elements we added takes up the same space in the column.

Bootstrap supports a maximum of 12 colums in every container. Bootstrap uses specific classes to control how much spacing each column uses.

And because we just only used "col", it means that every single element will get an equare share of the 12 columns available. Since I had 3 columns inside this row, they will each get 12/3 = 4 column spacing each
